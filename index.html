상세 내용

· 프로젝트 과정

안드로이드 스튜디오 활용한 페이지 제작 -> 아두이노 연결 확인 -> 파이썬과 아두이노 연동 -> 데이터 실시간 출력 확인 -> 장고 환경설정 -> Channels 라이브러리 추가 -> 안드로이드 스튜디오와 파이썬 연결 -> 파이썬 화면 크롤링

· 한계점

안드로이드 스튜디오에서 파이썬에 연결된 아두이노의 데이터 값을 가져와서 화면에 출력하려 했으나 소켓통신을 사용하는 과정에서 많은 시간이 소요되었고, 학술대회라는 포커스에 맞춰서 논문 작업을 병행해야 했기에 디테일을 살리지 못했다.

· 배운점

Django Channels는 Django 애플리케이션에 비동기 기능을 추가하고, WebSocket과 같은 실시간 기능을 구현할 수 있다. Daphne는 Django Channels와 함께 사용되는 ASGI 서버로, Django 애플리케이션에서 비동기 프로토콜을 처리할 수 있다.
